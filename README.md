# ✝️ Studio Bíblico — Deus Presente

Aplicativo bíblico para jovens (10-15 anos) com foco pentecostal.

## 🚀 Como usar

1. Abra o arquivo `App.jsx` no Claude.ai como React Artifact
2. Entre com o e-mail de admin: `admin@studiobiblico.com`
3. No Painel Admin, libere os e-mails dos usuários

## 📋 Funcionalidades

- 🔐 **Login com controle de acesso por e-mail** — admin libera quem entra
- 📖 **Estudo por Livro** — 66 livros, capítulos, recursos, versão bíblica
- 📅 **Plano de Leitura** — 6 planos prontos com guia motivacional
- 🎯 **Quiz Bíblico** — IA gera perguntas com 4 níveis e feedback
- 🕯️ **Devocional Diário** — estilo jovem + narração por voz
- ✍️ **Esboço de Sermão** — esboço completo com pontos e aplicação
- ⚙️ **Painel Admin** — libera/remove e-mails com persistência
- 📖 **Painel de Versículos** — clique em qualquer referência bíblica
- 📄 **Baixar PDF** — em todos os estudos gerados

## 🎨 Estilo

- Layout branco limpo com tipografia Nunito + Lora
- Seções numeradas animadas com emojis (estilo devocional jovem)
- Linguagem jovem, gírias cristãs, pentecostal
- Cores: roxo (#7C3AED) como cor principal

## 🔑 Acesso Admin

- E-mail: `admin@studiobiblico.com`
- O admin pode liberar qualquer e-mail pelo Painel Admin

## 🛠️ Tecnologias

- React (JSX) com hooks
- API Anthropic Claude Sonnet para geração de conteúdo
- window.storage para persistência de e-mails
- SpeechSynthesis para narração de áudio
- window.open + print para exportação PDF
